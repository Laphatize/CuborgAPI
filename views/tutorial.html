Verification Tutorial

A simple guide to verify your Cubash account through Cuborg.

Step 1:

Enter your username into the input box.

Step 2:

Copy the code provided.

Step 3:

Paste the code in the top of your bio. Make sure you leave a line of space underneath the code.

Step 4:

If you did everything correctly, you will see this screen.

Step 5:

Type !verify in Discord and Cuborg should response something like:

Nice job! You completed this tutorial!

Please note you'll have to create a Cubash account at cubash.com to use Cuborg.
